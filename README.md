# Expensify

> Track your expenses with ease.

## Run locally
This project uses yarn.

```bash
git clone https://github.com/akrabdev/expensify-app
cd expensify-app
yarn 
yarn dev-server
```
## Features: 
- Authentication using Google
- Add, Edit, Remove expenses
- Filter expenses to view on Dashboard
- Get a quick summary of your expenses
- Responsive on all devices
